0미전연 법카 Trace란?

법인카드를 사용한 뒤 카드 승인 문자·카카오톡 알림을 붙여넣거나, 직접 입력하거나, 영수증 사진을 찍으면 가맹점·금액·일시·분류가 자동으로 정리되어 클라우드에 저장되는 개인용 지출 기록 앱입니다.

  • 설치 불필요 — 인터넷 주소로 접속해서 바로 사용 (홈 화면 바로가기 추천)
  • 휴대폰을 바꾸거나 브라우저 기록을 지워도 로그인만 하면 내역이 그대로 유지됩니다.
  • 본인이 입력한 내역은 본인만 볼 수 있습니다.
  • 월별 계정(분류)별 예산과 사용액을 관리할 수 있습니다.
접속 주소: https://sitacule.github.io/card/ — 이 주소를 즐겨찾기/홈 화면에 추가해 두세요.

1회원가입

  1. 휴대폰 또는 PC 브라우저에서 https://sitacule.github.io/card/ 에 접속합니다.
  2. 로그인 화면 아래의 계정이 없으신가요? 회원가입 을 누릅니다.
  3. 회사 이메일을 입력합니다. 반드시 @lotte.net 주소만 가입할 수 있습니다.
  4. 비밀번호를 입력합니다. (6자 이상)
  5. 회원가입 버튼을 누르면 가입과 동시에 로그인되어 바로 사용할 수 있습니다.
@lotte.net 이 아닌 이메일은 가입·로그인이 되지 않습니다. 회사 메일 계정으로 진행해 주세요.

2로그인 / 비밀번호 찾기

로그인

  1. 가입한 회사 이메일과 비밀번호를 입력합니다.
  2. 로그인 버튼을 누르면 앱으로 들어갑니다.
한 번 로그인하면 같은 기기·브라우저에서는 로그인 상태가 유지됩니다. 공용 PC에서는 사용 후 우측 상단 로그아웃 을 눌러 주세요.

비밀번호를 잊었을 때

  1. 로그인 화면에서 비밀번호를 잊으셨나요? 를 누릅니다.
  2. 이메일 칸에 본인 회사 이메일을 입력한 상태로 누르면, 재설정 메일이 발송됩니다.
  3. 메일의 링크를 눌러 새 비밀번호를 설정한 뒤 다시 로그인합니다.

3휴대폰 홈 화면에 바로가기 추가

홈 화면에 추가하면 앱처럼 아이콘을 눌러 바로 실행할 수 있어 편리합니다. 운영체제별로 방법이 다릅니다.

🤖 Android (크롬 Chrome)
  1. 크롬(Chrome) 으로 https://sitacule.github.io/card/ 에 접속합니다.
  2. 화면 우측 상단의 점 3개 ⋮ 메뉴를 누릅니다.
  3. 홈 화면에 추가 (또는 앱 설치) 를 선택합니다.
  4. 이름을 확인하고 추가 를 누릅니다.
  5. 홈 화면에 생긴 미전연 법카 Trace 아이콘을 눌러 바로 접속합니다.
하단에 "홈 화면에 추가" 안내 배너가 자동으로 뜨면 그걸 눌러도 됩니다.
🍎 iPhone · iPad (사파리 Safari)
  1. 반드시 사파리(Safari) 로 https://sitacule.github.io/card/ 에 접속합니다.
    ※ 크롬 등 다른 브라우저에서는 홈 화면 추가가 안 됩니다.
  2. 화면 하단(또는 상단)의 공유 버튼 ⬆️ (네모에서 위로 향한 화살표) 을 누릅니다.
  3. 메뉴를 아래로 내려 홈 화면에 추가 를 선택합니다.
  4. 이름을 확인하고 우측 상단 추가 를 누릅니다.
  5. 홈 화면에 생긴 미전연 법카 Trace 아이콘을 눌러 바로 접속합니다.
iPhone은 사파리에서만 홈 화면 추가가 가능합니다. 메일/메신저에서 링크를 열면 다른 브라우저로 열릴 수 있으니, 주소를 복사해 사파리에 직접 붙여넣어 주세요.

4사용 내역 입력 (3가지 방법)

하단 메뉴 입력 화면에서 아래 3가지 중 편한 방법으로 내역을 추가합니다. 어떤 방법이든 마지막에 내용을 확인하고 저장하기 를 누르면 됩니다.

① 카드 승인 문자 / 카카오톡 붙여넣기 가장 정확 · 추천

  1. 카드사에서 온 승인 문자 또는 카카오톡 알림 메시지를 전체 복사합니다.
  2. 앱의 카드 승인 문자 / 카카오톡 붙여넣기 칸에 붙여넣습니다.
  3. 📋 붙여넣은 내역 인식 을 누르면 가맹점·금액·일시·카드가 자동으로 채워집니다.
예) 롯데백화점 잠실점 / 39,800원 승인 / 조*준 롯데법인카드(8382) / 일시불, 06/17 11:28 → 가맹점·금액·시간·카드가 자동 입력됩니다.

② 직접 입력

  1. ✏️ 사진 없이 직접 입력 을 누릅니다.
  2. 가맹점·금액·일자/시간 등을 직접 입력합니다. (일자·시간은 현재 시각으로 자동 입력됩니다.)

③ 영수증 사진

  1. 영수증 촬영(카메라) 또는 사진 선택(갤러리) 중 하나를 누릅니다.
  2. 영수증이 인식되어 글자가 자동으로 채워집니다. (처음 한 번은 인식 엔진을 내려받아 시간이 걸립니다.)
영수증 사진 인식은 감열지·흐림에 따라 정확도가 낮을 수 있습니다. 카드 문자 붙여넣기(①) 가 가장 정확합니다.

공통 — 내용 확인 후 저장

  1. 가맹점·금액·사용일자·사용시간이 맞는지 확인하고, 틀리면 수정합니다.
  2. 분류를 선택합니다: 복리_식대(내부) / 복리_업무활동(외부) / 교통비 / 기타.
    · 가맹점에 택시 등 교통 관련 단어가 있으면 자동으로 교통비로 잡힙니다.
  3. 외부(업무활동) 인 경우 상대 회사명과 동석자를 반드시 입력합니다.
  4. 저장하기 를 누르면 완료됩니다. (금액은 자동으로 콤마가 표시됩니다.)

5내역 조회 · 수정 · 내보내기

하단 메뉴 내역 에서 저장한 내역을 관리합니다.

기능방법
검색상단 검색창에 가맹점·메모·동석자·회사명 입력
월 필터 / 정렬월 선택, 최신순·금액순 등으로 정렬
수정항목의 ✏️ 수정 → 내용 변경 후 수정 저장
삭제 / 되돌리기삭제 후 잠깐 뜨는 실행취소 로 복구 가능
엑셀로 내보내기📤 CSV (엑셀에서 열 수 있음)
전체 백업💾 전체 백업(JSON) (사진 포함 백업)

상단에는 총 건수 · 총 금액 · 이번달 금액이 표시됩니다.

6예산 관리

하단 메뉴 예산 관리 에서 월별·분류별 예산을 관리합니다.

  1. 상단에서 월을 선택합니다. (기본: 이번 달)
  2. ✏️ 수정 을 눌러 입력을 활성화합니다.
  3. 분류별 예산 금액을 입력합니다. (천단위 콤마 자동 표시)
  4. 저장 을 누르면 예산이 저장됩니다.
  5. 분류별로 사용액 · 사용률 막대 · 잔액이 표시되고, 맨 아래 총 사용액을 확인할 수 있습니다.
↩ 지난달 예산 불러오기 로 매달 반복 입력을 줄일 수 있습니다.

7자주 묻는 질문

Q. 휴대폰을 바꾸면 내역이 사라지나요?

아니요. 내역은 클라우드에 저장되므로, 새 기기에서 같은 계정으로 로그인하면 그대로 나타납니다.

Q. 브라우저 기록(캐시)을 지우면 데이터가 사라지나요?

사라지지 않습니다. 다시 로그인하면 복원됩니다.

Q. 다른 사람이 내 내역을 볼 수 있나요?

볼 수 없습니다. 본인 계정의 내역은 본인만 접근할 수 있습니다.

Q. 인터넷이 안 되는 곳에서도 쓸 수 있나요?

이전에 한 번 접속했다면 오프라인에서도 보기·입력이 가능하며, 인터넷이 연결되면 자동으로 동기화됩니다. (단, 영수증 사진 첫 인식은 인터넷이 필요합니다.)

Q. 영수증 사진 인식이 자꾸 틀려요.

카드 승인 문자/카카오톡 붙여넣기가 가장 정확합니다. 그 방법을 우선 사용해 주세요.
