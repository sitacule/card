0미전연 법카 Trace란?

법인카드를 사용한 뒤 카드 승인 문자·카카오톡 알림을 붙여넣거나, 직접 입력하거나, 영수증 사진을 찍으면 가맹점·금액·일시·분류가 자동으로 정리되어 클라우드에 저장되는 개인용 지출 기록 앱입니다.

  • 설치 불필요 — 인터넷 주소로 접속해서 바로 사용 (홈 화면 바로가기 추천)
  • 휴대폰을 바꾸거나 브라우저 기록을 지워도 로그인만 하면 내역이 그대로 유지됩니다.
  • 본인이 입력한 내역은 본인만 볼 수 있습니다.
  • 월별 분류별 예산과 사용액을 관리할 수 있습니다.
접속 주소: https://sitacule.github.io/card/ — 이 주소를 즐겨찾기/홈 화면에 추가해 두세요.

1회원가입 (이메일 인증)

  1. https://sitacule.github.io/card/ 접속
  2. 로그인 화면에서 계정이 없으신가요? 회원가입 누르기
  3. 회사 이메일 입력 @lotte.net
  4. 비밀번호 입력 (6자 이상)
  5. 회원가입 버튼 누르기
📩 이메일 인증 — 회원가입 후 회사 이메일로 인증 메일이 발송됩니다. 메일의 인증 링크를 한 번 눌러 두시면 계정 인증이 완료됩니다. (메일이 안 보이면 스팸함도 확인해 주세요.)

2로그인 / 비밀번호 찾기

로그인

  1. 가입한 회사 이메일과 비밀번호 입력
  2. 로그인 버튼 누르기 → 앱 화면으로 이동
한 번 로그인하면 같은 기기에서는 로그인 상태가 유지됩니다. 공용 PC에서는 사용 후 우측 상단 로그아웃 을 눌러 주세요.

비밀번호를 잊었을 때

  1. 로그인 화면에서 비밀번호를 잊으셨나요? 누르기
  2. 이메일을 입력한 상태로 누르면 재설정 메일이 발송됩니다.
  3. 메일의 링크에서 새 비밀번호를 설정한 뒤 다시 로그인

3휴대폰 홈 화면에 바로가기 추가

홈 화면에 추가하면 앱처럼 아이콘을 눌러 바로 실행할 수 있어 편리합니다. 운영체제별로 방법이 다릅니다.

🤖 Android (크롬 Chrome)
  1. 크롬(Chrome) 으로 sitacule.github.io/card 접속
  2. 우측 상단 점 3개 ⋮ 메뉴 누르기
  3. 홈 화면에 추가 (또는 앱 설치) 선택
  4. 이름 확인 후 추가
  5. 홈 화면의 미전연 법카 Trace 아이콘으로 바로 접속
🍎 iPhone · iPad (사파리 Safari)
  1. 반드시 사파리(Safari) 로 sitacule.github.io/card 접속
    ※ 크롬 등 다른 브라우저에서는 추가가 안 됩니다.
  2. 하단(또는 상단)의 공유 버튼 ⬆️(네모+위 화살표) 누르기
  3. 메뉴를 내려 홈 화면에 추가 선택
  4. 우측 상단 추가
  5. 홈 화면의 미전연 법카 Trace 아이콘으로 바로 접속
메일/메신저의 링크는 사파리가 아닌 앱으로 열릴 수 있습니다. 주소를 복사해 사파리에 직접 붙여넣어 주세요.

4사용 내역 입력 (3가지 방법)

하단 메뉴 입력 에서 아래 3가지 중 편한 방법으로 추가하고, 마지막에 저장하기 를 누릅니다.

① 카드 문자 붙여넣기 가 가장 정확합니다. ② 직접 입력, ③ 영수증 촬영/사진 선택도 가능합니다.

① 카드 승인 문자 / 카카오톡 붙여넣기 가장 정확 · 추천

  1. 카드사 승인 문자 또는 카카오톡 알림을 전체 복사
  2. 앱의 붙여넣기 칸에 붙여넣기
  3. 📋 붙여넣은 내역 인식 → 가맹점·금액·일시·카드가 자동 입력
예) 롯데백화점 잠실점 / 39,800원 승인 / 조*준 롯데법인카드(8382) / 일시불, 06/17 11:28 → 자동 입력됩니다.

② 직접 입력

  1. ✏️ 사진 없이 직접 입력 누르기
  2. 가맹점·금액 입력 (일자·시간은 현재 시각으로 자동 입력)

③ 영수증 사진

  1. 영수증 촬영(카메라) 또는 사진 선택(갤러리) 누르기
  2. 영수증 글자가 자동 인식됩니다. (처음 한 번은 인식 엔진 다운로드로 시간이 걸립니다.)

공통 — 내용 확인 후 저장

  1. 가맹점·금액·사용일자·사용시간 확인 후 틀리면 수정
  2. 분류 선택: 복리_식대(내부) / 복리_업무활동(외부) / 교통비 / 기타
    · 가맹점에 택시 등 교통 관련 단어가 있으면 자동으로 교통비로 잡힙니다.
  3. 외부(업무활동) 인 경우 상대 회사명·동석자 입력
  4. 저장하기 (금액은 자동으로 콤마 표시)

5내역 조회 · 수정 · 내보내기

하단 메뉴 내역 에서 저장한 내역을 관리합니다.

기능방법
검색상단 검색창에 가맹점·메모·동석자·회사명 입력
월 필터 / 정렬월 선택, 최신순·금액순 등으로 정렬
수정항목의 ✏️ 수정 → 변경 후 수정 저장
삭제 / 되돌리기삭제 후 잠깐 뜨는 실행취소 로 복구 가능
엑셀로 내보내기📤 CSV (엑셀에서 열림)
전체 백업💾 전체 백업(JSON) (사진 포함)

상단에 총 건수 · 총 금액 · 이번달 금액이 표시됩니다.

6예산 관리

하단 메뉴 예산 관리 에서 월별·분류별 예산을 관리합니다.

  1. 상단에서 월 선택 (기본: 이번 달)
  2. ✏️ 수정 을 눌러 입력 활성화
  3. 분류별 예산 금액 입력 (천단위 콤마 자동)
  4. 저장 누르기
  5. 분류별 사용액 · 사용률 막대 · 잔액과 맨 아래 총 사용액 확인
↩ 지난달 예산 불러오기 로 매달 반복 입력을 줄일 수 있습니다.

7자주 묻는 질문

Q. 휴대폰을 바꾸면 내역이 사라지나요?

아니요. 클라우드에 저장되므로 새 기기에서 같은 계정으로 로그인하면 그대로 나타납니다.

Q. 브라우저 기록(캐시)을 지우면 사라지나요?

사라지지 않습니다. 다시 로그인하면 복원됩니다.

Q. 다른 사람이 내 내역을 볼 수 있나요?

볼 수 없습니다. 본인 계정의 내역은 본인만 접근할 수 있습니다.

Q. 인터넷이 안 되는 곳에서도 쓸 수 있나요?

이전에 접속한 적이 있으면 오프라인에서도 보기·입력이 가능하고, 연결되면 자동 동기화됩니다. (영수증 사진 첫 인식은 인터넷 필요)

Q. 영수증 사진 인식이 자꾸 틀려요.

카드 승인 문자/카카오톡 붙여넣기가 가장 정확합니다. 그 방법을 우선 사용해 주세요.
